# Exposing SCCM and MSSQL Attack Paths in Hardened Environments with OpenGraph

**Speaker:** Chris Thompson

## Abstract

Struggling to find attack paths in Active Directory environments where all the obvious BloodHound edges have been mitigated? This talk introduces two new OpenGraph collectors that expose lesser-known SCCM and MSSQL attack chains starting with unauthenticated network access to gaining administrative control of the environment.

We'll walk through collection and graph visualization of these attack paths, demonstrate a full attack chain, and show you how to operationalize this tooling in real engagements. If you're tired of hitting the same hardened defenses, this talk will help you uncover blind spots that defenders may have overlooked.
