# Mapping Shadows in the Cloud: Exploring Google Cloud Attack Paths with GCP-Hound and OpenGraph

**Speaker:** Faiz Karim

## Abstract

The presentation explores the practical architecture of GCP-Hound, featuring modular collectors for real-world GCP resources, custom edge types such as CanImpersonate and CanReadSecrets, and direct export to BloodHound/OpenGraph. While the tool is actively evolving, the presentation will walk attendees through practical examples that show how to map and query GCP attack paths using the tool. The session covers ongoing development, identifies current limitations, and outlines plans for continued improvement, emphasizing exploration and learning within the security research community.

Whether the attendees use GCP as a red teamer, defender or just to expand their attack path toolkit, this session will equip them with practical techniques, sharp insights, and a solid foundation for graphing cloud identities and uncovering real attack paths in Google Cloud Platform.
