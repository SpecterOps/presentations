# CyberArkHound: Integrating CyberArk Privileged Access into BloodHound Attack Path Analysis

**Speaker:** Javier Nestor Azofra Ovejero and Julian Garcia Murias

## Abstract

Organizations invest heavily in both Active Directory security monitoring (BloodHound) and Privileged Access Management (CyberArk), yet these systems operate in silos. This session unveils a novel integration that exports CyberArk vault data into BloodHound's OpenGraph format, enabling security teams to visualize complete attack paths spanning both AD and privileged credential systems. Attendees will learn how to identify critical exposure where compromised AD accounts have access to privileged credentials in CyberArk safes, creating invisible escalation paths traditional tools miss. The presentation includes a live demonstration of the open-source exporter tool, real-world use cases from enterprise security assessments, and practical Cypher queries for hunting privilege escalation opportunities. This integration transforms BloodHound from an AD/EntraID tool into a comprehensive privileged access risk analyzer, helping SOCs prioritize remediation based on actual credential access chains.
