# AdminSDHolder Misconceptions & Misconfigurations

**Speaker:** Jim Sykora

## Abstract

This talk will distill my 159-page, 420 minute read, E-Book on AdminSDHolder and the associated Active Directory and Windows internals down into a short talk. We'll dive deep into the Microsoft Windows access control model, Active Directory history, the AdminSDHolder protection mechanism, what SDProp actually does, and some of the common mistakes that keep popping up in AD Forests everywhere.
