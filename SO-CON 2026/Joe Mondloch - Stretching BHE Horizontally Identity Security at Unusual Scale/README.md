# Stretching BHE Horizontally: Identity Security at Unusual Scale

**Speaker:** Joe Mondloch

## Abstract

What’s a “big” BloodHound Enterprise environment? Is it a single Active Directory domain with a million accounts—or hundreds of domains, each with only a few thousand?

In this talk, we’ll dive into how a large healthcare software company and its hosting division use BloodHound Enterprise at scale. Our environment includes hundreds of Active Directory domains and nearly a thousand SharpHound clients—an architecture that pushes BHE (and other security tools) in unique ways.

We’ll explore how we operate BloodHound Enterprise in this kind of distributed environment, how we use it to audit a complex multi-tier account and access model, and how it plays a critical role in ensuring security controls are applied consistently across domains. We’ll show how this continuous validation has helped us uncover tiering violations, access over-provisioning, and other common mistakes. Attendees will leave with practical insights from scaling identity security tooling in an unusual environment, and how those lessons apply to more traditional ones as well.
