# Your CI/CD Pipeline is My Attack Path: Graphing GitHub OIDC to Cloud Takeover

**Speaker:** Julian Catrambone

## Abstract

No credentials to steal? No problem. This talk shows how attackers can exploit OIDC federation to move from a GitHub fork to full AWS or Azure access through misconfigured trust policies. We'll demonstrate the attack live, then show defenders how to use BloodHound collectors and the OpenGraph platform to discover these hidden cross-platform identity chains before red teams (or real attackers) do.
