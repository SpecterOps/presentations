# The Identity APM Maturity Model: A Framework for Leveling Up

**Speaker:** Kate Dawson Nathan Davis and Hugo van den Toorn

## Abstract

This panel brings together identity and security practitioners to walk through the Identity APM Maturity Model as a practical framework for growth. Panelists will discuss what each stage looks like in the real world, common pitfalls teams face as they mature, and how to prioritize next steps—whether you’re just getting started or scaling an established Attack Path Management program.
