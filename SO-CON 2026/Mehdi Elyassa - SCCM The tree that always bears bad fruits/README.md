# SCCM: The tree that always bears bad fruits

**Speaker:** Mehdi Elyassa

## Abstract

Microsoft Configuration Manager (SCCM) is a high‑value but often overlooked target, capable of executing code on managed clients and storing sensitive credential data. In this session, I’ll present newly discovered attack paths and weaknesses that allow escalation from low‑privileged or unauthenticated access to full control of the SCCM hierarchy, and reveal a new post‑exploitation technique that leverages SCCM’s architecture to operate beyond normal database connectivity constraints. Attendees will gain deep insight into SCCM’s architecture and internals and learn actionable strategies for detecting and hardening these systems. This talk assumes familiarity with SCCM fundamentals and focuses on advanced internals and exploitation techniques.
