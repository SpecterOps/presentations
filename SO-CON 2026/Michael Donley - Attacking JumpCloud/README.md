# Attacking JumpCloud

**Speaker:** Michael Donley

## Abstract

It's become increasingly common to find organizations trying to ditch traditional Active Directory management in favor of cloud-based unified identity platforms. JumpCloud has emerged as a popular choice for managing the modern heterogeneous enterprise: one console to control Windows, macOS, Linux, iOS, and Android devices, with identity management, SSO, and MFA all bundled together. Deploy a lightweight agent on your endpoints, and let the cloud handle the rest...

In this talk, we'll walk through the main attack surface present in a JumpCloud tenant. We'll look at why JumpCloud's admin portal should be protected with the same paranoia you'd apply to Domain Admin accounts, explore how attackers can pivot from a compromised endpoint to the cloud control plane, and examine the offensive opportunities that arise from the MDM enrollment workflows.
