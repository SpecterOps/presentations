# Mapping the Adversary: Enriched Incident Graphs with BloodHound Data in Kusto

**Speaker:** Olaf Hartong

## Abstract

BloodHound has been the de facto tool for attack path discovery.  Properly generated alerts and incidents can show interesting graphs as well. When enriched and graphed properly, this can uncover many interesting correlations. When we start combining these graphs in our security monitoring program is when it becomes truly magical.
