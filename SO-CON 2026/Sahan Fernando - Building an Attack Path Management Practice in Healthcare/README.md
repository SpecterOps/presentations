# Building an Attack Path Management Practice in Healthcare

**Speaker:** Sahan Fernando

## Abstract

While the theories and discoveries in attack path management have shown significant risk, there is always the what is next step. This session will discuss the build and ongoing operational workings of attack path management in an enterprise healthcare organization. Hear about how we have approached staffing, looked at KPI/KRI metrics for stakeholder reporting, and the challenges in moving the boulder uphill when collaborating with multiple teams on complex remediations.
