# AnsibleHound: Attack Path Management applied to Ansible

**Speaker:** Simon Lachkar and Charl-Alexandre Le Brun

## Abstract

The OpenGraph release in summer 2025 gave the community the ability to model attack paths on new platforms. This presentation will focus on AnsibleHound, a collector that adds Ansible WorX and Ansible Tower attack paths to BloodHound. Additionally, we will conduct a thorough exploration of Ansible exploitation and abuse through attack path management. This will enable both attackers and defenders to identify hybrid attack paths.
